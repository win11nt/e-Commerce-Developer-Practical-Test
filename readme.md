# e-Commerce Developer Practical Test

A simple front-end e-commerce project built with HTML, Tailwind CSS, and JavaScript, featuring responsive design for mobile and tablet.

## Tech Stack
- HTML5
- CSS3, Tailwind CSS
- JavaScript(ES6+)
- Font Awesome

## How to Run
1. Clone the repository:
```bash
git clone https://github.com/win11nt/e-Commerce-Developer-Practical-Test.git
```
2. Open index.html in your browser (or use Live Server in VS Code).

## Notes
Fully static front-end project

No backend or framework required
